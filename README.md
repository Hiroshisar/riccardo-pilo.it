# Personal Portfolio Website

This is a personal portfolio website built with modern frontend technologies.  
The project focuses on performance, responsiveness, and clean UI/UX design.

## 🚀 Tech Stack

### Core
- React 19
- TypeScript
- Vite

### Styling
- Tailwind CSS
- Tailwind Vite Plugin

### Animations
- Framer Motion

### Routing
- React Router DOM

### Icons
- React Icons (Font Awesome 6)

---

## 📦 Main Dependencies

- `react` – UI library
- `react-dom` – React DOM renderer
- `react-router-dom` – Client-side routing
- `framer-motion` – Animations and transitions
- `react-icons` – Icon library (Font Awesome 6)
- `@tailwindcss/vite` – Tailwind integration for Vite
- `tailwindcss` – Utility-first CSS framework

---

## 🛠️ Dev Tools

- TypeScript – Static typing
- Vite – Build tool and dev server
- ESLint – Code linting
- Prettier – Code formatting
- eslint-plugin-react-hooks – React hooks lint rules
- eslint-plugin-react-refresh – Fast refresh support
- prettier-plugin-tailwindcss – Tailwind class sorting

---

## 📁 Project Scripts

```bash
npm run dev       # Start development server
npm run build     # Build production bundle
npm run preview   # Preview production build
npm run lint      # Run ESLint
````

---

## 🌐 Features

* Fully responsive design
* Multi-page SPA using React Router
* Smooth animations with Framer Motion
* Modular and scalable component structure
* Clean and modern UI
* Optimized production build with Vite

---

## 📌 Deployment Notes

This project uses client-side routing.
If deployed on static hosting (e.g. Netlify), ensure proper redirect rules are configured:

```txt
/*  /index.html  200
```
